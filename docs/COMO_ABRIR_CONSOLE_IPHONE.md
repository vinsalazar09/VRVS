# Como Abrir Console do Safari no iPhone

## Passo a Passo

1. **Conectar iPhone ao Mac via cabo USB**
   - Use o cabo Lightning/USB-C

2. **No iPhone:**
   - Vá em Ajustes → Safari → Avançado
   - Ative "Inspeção Web"

3. **No Mac:**
   - Abra o Safari (não Chrome)
   - Menu: Desenvolver → [Nome do seu iPhone] → [Nome do app/site]
   - Isso abre o console do Safari

4. **Se não aparecer "Desenvolver":**
   - Safari → Preferências → Avançado
   - Marque "Mostrar menu Desenvolver na barra de menus"

## Alternativa: Console no próprio iPhone

Se não conseguir conectar:
- No iPhone, abra o app
- Toque e segure em qualquer lugar da página
- Escolha "Inspecionar Elemento" (se disponível)
- Ou use ferramentas de debug remoto

## O que procurar no console:

- Erros JavaScript (vermelho)
- Logs `[PENDENCIAS]` quando você toca nos itens
- Se `togglePendencia` está sendo chamado ou não
